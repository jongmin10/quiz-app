`src/components/` 폴더 안의 React 컴포넌트 전체를 읽고, 아래 기준으로 리뷰해줘.

## 리뷰 기준

### 1. 코드 품질
- 불필요한 re-render를 유발하는 코드가 있는가?
- `useEffect` 의존성 배열이 올바른가?
- `useRef` / `useCallback` / `useMemo` 사용이 적절한가?
- 중복 코드나 추출 가능한 로직이 있는가?

### 2. 접근성 (a11y)
- 모든 인터랙티브 요소에 `aria-label` 또는 명확한 텍스트가 있는가?
- 키보드로 모든 기능을 사용할 수 있는가?
- `role`, `aria-live`, `aria-pressed` 등 ARIA 속성이 올바르게 사용되는가?
- 색상 외에 아이콘·텍스트로도 상태를 구분할 수 있는가?

### 3. 성능
- 컴포넌트가 필요 이상으로 크지 않은가? (분리 고려)
- 이벤트 리스너가 cleanup 없이 등록되는 경우는 없는가?
- 불필요한 상태(state)가 있는가?

### 4. UX
- 에러 상태·로딩 상태가 명확하게 처리되는가?
- 사용자 피드백(성공·실패 메시지)이 충분한가?

## 출력 형식

컴포넌트별로:
- 🔴 **즉시 수정** — 버그·접근성 오류
- 🟡 **개선 권장** — 성능·코드 품질
- 🟢 **양호**

지적 사항이 있으면 **수정 코드와 함께** 제안해줘.
